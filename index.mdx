## What BBOT does

BBOT seeds a scan with one or more targets — domains, IPs, CIDR ranges, URLs, emails, GitHub orgs — then spiders outwards recursively. Each module consumes events from other modules and produces new ones, creating a self-reinforcing web of discovery.

```bash
# Find subdomains of evilcorp.com
bbot -t evilcorp.com -p subdomain-enum

# Run a full web scan
bbot -t www.evilcorp.com -p web-thorough

# Scan everything, everywhere, all at once
bbot -t evilcorp.com -p kitchen-sink --allow-deadly
```
